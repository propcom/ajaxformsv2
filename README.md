ajaxformsv2
===========

Ajax Forms v2 (jQuery Plugin)

Include ajax-forms.js in your js/libs directory and reference in template.php, then initialize like so;-

$('#frm-test').ajaxform({
  	'error_location' : 'label',
  	'show_alert'     : false
})

(See script.js)

Working example at;-

http://ajaxforms.anthony.local/
